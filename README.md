## ðŸ¤¸ Getting Started

#### From your command line:

##### 1. Clone this repository

```bash
    $ git clone https://github.com/Junmarkchi97/bukid-markets
```

##### 2. Go into folder

```bash
    $ cd bukid-markets
```

##### 3. Install needed gems

```ruby
    $ bundle install
```

##### 4. Setup Database

```ruby
    $ rails db:setup
```

##### 5. Start the application

```ruby
    $ rails server
```
