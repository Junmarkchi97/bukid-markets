## Getting Started

From your command line:

```
# Clone this repository
$ git clone https://github.com/Junmarkchi97/bukid-markets

# Go into repository
$ cd bukid-markets

# Install needed gems
$ bundle install

# Setup Database
$ rails db:setup

# Run the server
$ rails server
```
